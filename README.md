# Goddess-Bot


## Setup

```bash
git clone https://github.com/Red-Cafe/Goddess-Bot.git Goddess-Bot

cd Goddess-Bot

npm install
```

Open config.json, and change everything

```json
{
    "port": "8080",
    "token": "your-token",
    "ownerID": "your-id",
    "guildID": "your-guild-id",
    "clientID": "your-client-id",
}
```



## Building

```bash

node deploy.js

node .
```
